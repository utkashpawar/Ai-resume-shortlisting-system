# AI Resume Shortlisting System

## Overview

This project helps recruiters quickly analyze resumes and compare them with job descriptions.

The system calculates a skill match score and identifies missing skills. It also integrates with Google Gemini AI to generate advanced analysis.

## Features

* Resume upload (.txt)
* Resume text input
* Skill matching system
* Match percentage progress bar
* Missing skill recommendations
* Gemini AI integration

## Technologies Used

* HTML
* CSS
* JavaScript
* Google Gemini AI

## How It Works

1. Upload or paste a resume.
2. Paste the job description.
3. Click "Analyze Resume".
4. The system calculates a match score and shows missing skills.
5. Use "Analyze with Gemini AI" for deeper analysis.

## Future Improvements

* PDF resume parsing
* Automatic skill extraction using AI
